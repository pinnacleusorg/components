# Introduction

QUISP (Quick UI for Svelte at Pinnacle) is a UI component library written using Svelte, which can be consumed by other Svelte projects in the organization. It is designed to house all reused and potentially reusable components across all of our projects, with the ability to modify styles going forward. The library is aligned with Pinnacle's semver system, where v3.0.0 corresponds to Pinnacle's 2023 event.

Features:

- A collection of reusable, flexible branding-approved UI components
- Easy to update styles for new and old projects
- Versioned using Pinnacle's aligned semver system

<br />

## Getting Started

<br />

### Installation

QUISP is available as an npm package.

```bash
// with npm
npm install @pinnacleusorg/quisp

// with yarn
yarn add @pinnacleusorg/quisp
```

### Usage

QUISP is designed to be consumed by other Svelte projects. To use a component, simply import the component you want to use from the package.

```svelte
<script lang="ts">
	import { Button } from '@pinnacleusorg/quisp';
</script>
```

Component names directly align to the names presented in the component list in the sidebar.
